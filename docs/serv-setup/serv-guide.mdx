# Путеводитель по серверу

Путеводитель по серверу — это специальная страница, которую новые участники увидят после того, как ответят на вопросы, предшествующие присоединению, в процессе адаптации. Его цель — дать возможность администраторам серверов настроить приветствие для пользователей, предлагая увлекательное и информативное введение в своё сообщество.

Используя Путеводитель по серверу, администраторы могут продемонстрировать уникальные функции, правила и каналы своего сервера, чтобы новички чувствовали, что им здесь рады, и получили всю информацию с момента их присоединения. Такой индивидуальный подход помогает новым пользователям быстро адаптироваться на сервере.

## Как настроить путеводитель по серверу

![An image from the static](/img/serv-guide.jpg)

Путеводитель по серверу является частью функции «Адаптация», которая доступна через «Настройки сервера» и выберите раздел «Адаптация».

Как только вы завершите настройку каналов по умолчанию и раздела с вопросами для персонализации, вы сможете начать настройку Путеводителя по серверу. Здесь вы можете задать несколько вещей:

- Приветственный знак: место в верхней части Путеводителя по серверу, где вы можете написать персональное сообщение от себя, администратора.
- Список дел для новых участников: 3-5 заданий для новых участников, которые помогут им начать общаться в ваших каналах.
- Ресурсы: Превратите каналы только для чтения в красивые страницы с ресурсами в вашем Путеводителе по серверу. Они больше не будут появляться в списке каналов и будут иметь некоторые преимущества:
- Пользователи будут перенаправляться в верхнюю часть страниц, а не в конец ветки сообщений;
- Строки чата и аватары будут удалены, чтобы всё выглядело аккуратно;
- Содержимое, вложения, медиаконтент и форматирование останутся без изменений.
