# Путеводитель по серверу

**Путеводитель по серверу** — это специальная страница, которую новые участники увидят после того, как ответят на вопросы при присоединении, в процессе адаптации.
Его цель — предоставить администраторам возможность настроить приветствие, предлагая увлекательное и информативное введение в сообщество.

С помощью Путеводителя администраторы могут показать уникальные особенности,
правила и каналы сервера, чтобы новички почувствовали себя желанными участниками и получили всю необходимую информацию сразу после присоединения.
Такой подход помогает новым пользователям быстрее адаптироваться на сервере.

## Настроить путеводитель по серверу

![An image from the static](/img/serv-guide.jpg)

**Путеводитель по серверу** является частью функции «Адаптация», доступной через «Настройки сервера» в разделе «Адаптация».

После настройки каналов по умолчанию и раздела с вопросами для персонализации, вы можете перейти к настройке Путеводителя. Здесь можно настроить несколько важных элементов:

- **Приветственный знак**: в верхней части Путеводителя разместите персональное сообщение от себя, администратора.
- **Список дел для новых участников**: добавьте 3-5 заданий для новичков, чтобы помочь им начать общаться в каналах.
- **Ресурсы**: превратите каналы только для чтения в стильные страницы с ресурсами, которые будут отображаться в Путеводителе, а не в списке каналов.
- Пользователи будут перенаправляться в верхнюю часть страниц, а не в конец ветки сообщений.
- Строки чата и аватары будут удалены, чтобы интерфейс выглядел аккуратно.
- Содержимое, вложения, медиаконтент и форматирование останутся без изменений.
