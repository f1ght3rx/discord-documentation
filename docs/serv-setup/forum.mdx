# Форум

Форум-каналы создают пространство для упорядоченных дискуссий.
Позволяют обсуждать конкретные темы без риска перебивать других участников.
Благодаря организации обсуждений в формате публикаций легко возвращаться к существующим темам или начинать новые, без опасений, что сообщение останется незамеченным.

![An image from the static](/img/forum1.jpg)

## Как создать форум-канал

1. В списке каналов необходимо навести курсор на категорию,
где требуется создать форум-канал, и нажать кнопку «+»

![An image from the static](/img/channels.jpg)

В появившемся окне выбрать тип канала «Форум» и ввести название

2. Нажать кнопку «Создать канал»

![An image from the static](/img/forum3.jpg)

3. Нажмите кнопку «Создать канал»

## Права доступа

Сначала настраиваются права доступа к форум-каналу.
Эти разрешения определяют, кто сможет видеть и использовать канал.

![An image from the static](/img/1.jpg)

Для упрощения настройки можно нажать «Установить рекомендуемые разрешения» и ограничить доступ для определённых ролей.

## Правила публикации

После создания канала можно оформить рекомендации для новых публикаций.

![An image from the static](/img/1.jpg)

Нажав «Создать руководство к посту», открывается страница настроек,
где можно изменить название канала, добавить инструкции и настроить теги для удобства участников.

![An image from the static](/img/2.jpg)

## Теги

Теги помогают классифицировать публикации и могут быть обязательным условием для создания нового поста.

![An image from the static](/img/tags.jpg)

Если теги не были добавлены сразу, их можно установить позже:
для этого достаточно кликнуть правой кнопкой мыши по публикации и выбрать «Редактировать теги»

![An image from the static](/img/tags2.jpg)

## Установка макета по умолчанию

Для форум-канала можно выбрать один из двух макетов: «Вид списка» или «Вид галереи».
Выбор зависит от тематики обсуждений и желаемого формата представления информации.

![An image from the static](/img/3.jpg)

### Вид списка

Этот вариант идеально подходит для текстовых обсуждений и помогает пользователям легко присоединяться к беседе.

## Как искать публикации на форуме

1. В списке каналов найти нужный форум-канал
2. В строке поиска в верхней части страницы ввести ключевое слово или фразу из заголовка публикации.
Поисковая система отобразит релевантные результаты

## Фильтрация публикаций по тегам

![An image from the static](/img/6.jpg)

Для удобной навигации публикации можно фильтровать по тегам.
Также доступен вариант выбора нескольких тегов одновременно через пункт «Все», что позволяет точнее находить нужные посты.
