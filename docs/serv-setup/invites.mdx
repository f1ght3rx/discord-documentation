# Приглашения

## Принятие заявок на сервер

Пользователи могут присоединиться к серверу тремя способами:
- Только по приглашению
- Через путешествие
- Через заявку

Заявки на участие позволяют установить более точный контроль над доступом к серверу.
С их помощью можно просматривать запросы на вступление и принимать решение о допуске каждого пользователя.
Это обеспечивает повышенную безопасность и управляемость сообщества.

## Настройка доступа к серверу

1. Стоит выбрать вкладку «Настройки сервера»
![An image from the static](/img/access1.jpg)
2. Перейдите во вкладку «Доступ»
![An image from the static](/img/access2.jpg)

### Только по приглашению
Режим «Только по приглашению» — это стандартная настройка конфиденциальности для всех серверов Discord,
обеспечивающая максимальный контроль над тем, кто может присоединиться
- Доступ на сервер предоставляется только по действующим ссылкам-приглашениям
- Можно активировать правила сервера, чтобы новые участники подтверждали их перед входом
- Дополнительно можно создавать ссылки с ограниченным сроком действия и числом использований
![An image from the static](/img/access11.jpg)

### Присоединение по заявке
Опция «Присоединение по заявке» позволяет фильтровать новых участников с помощью формы, которую необходимо заполнить перед вступлением
- Каждая заявка требует одобрения администратором, прежде чем пользователь получит доступ к серверу
- Есть возможность задать собственные вопросы, чтобы лучше понять мотивацию и интересы потенциальных участников
- Ссылку на форму можно отобразить в профиле сервера, чтобы её было проще найти
![An image from the static](/img/access12.jpg)

### Публичный
Выбор настройки «Публичный» позволяет включить функцию «Путешествие по серверам»,
делая сервер доступным для всех пользователей Discord, заинтересованных в новых сообществах.
- Для этого сервер должен быть оформлен как сервер сообщества
- Необходимо соответствовать стандартам Discord, включая требования по безопасности
- Сервер становится доступным через «Путешествие по серверам», и любой пользователь может присоединиться
