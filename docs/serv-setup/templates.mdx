# Шаблоны

С помощью шаблонов сервера можно автоматически скопировать существующие категории, каналы, роли и разрешения.
Это позволяет значительно упростить процесс создания нового сервера.
Новый сервер будет иметь ту же структуру, что и исходный, без необходимости ручной настройки.

## Создание шаблона

1.  Для создания шаблона сервера требуется владение сервером или наличие роли с активированным разрешением «Управлять сервером»

![An image from the static](/img/templates1.jpg)

2.  Следующим шагом следует перейти в раздел настроек сервера,
выбрав соответствующий пункт в выпадающем меню рядом с именем сервера или щёлкнув правой кнопкой мыши по его названию,
перейти во вкладку «Шаблон сервера»

![An image from the static](/img/templates.jpg)

3.  После перехода в раздел «Шаблон сервера» предоставляется возможность указать название и краткое описание шаблона.
По завершении заполнения этих полей необходимо нажать кнопку «Создать шаблон» для генерации ссылки на шаблон
4.  После создания ссылки достаточно выбрать пункт «Копировать», чтобы получить возможность поделиться шаблоном сервера

## Синхронизация и удаление шаблонов сервера

Шаблоны серверов не обновляются автоматически при внесении изменений в настройках исходного сервера.
Однако при внесении изменений на сервере будет появляться уведомление под его именем

![An image from the static](/img/templates11.jpg)

Для синхронизации шаблона сервера необходимо выполнить следующие действия:
1.  Открыть раздел «Настройки сервера»
2.  Перейти во вкладку «Шаблоны сервера»
3.  Нажать кнопку «Синхронизировать шаблон» для обновления текущего шаблона на основе актуальной структуры сервера

![An image from the static](/img/templates12.jpg)

Для удаления сервера необходимо выполнить следующие действия:
1. Открыть «Настройки сервера»
2. Перейти во вкладку «Шаблоны сервера»
3. В нижней части раздела нажать кнопку «Удалить сервер»
4. Подтвердить удаление, следуя инструкциям, указанным в диалоговом окне
(может потребоваться ввести название сервера или подтвердить действие через код/двойную аутентификацию)
