# Создание сообщества

Зачем создавать сообщество:
1. Доступ к профессиональным инструментам модерации
    - Автомодерация — автоматическое удаление спама, оскорблений и запрещённого контента
    - Журнал аудита — отслеживание действий модераторов и участников
    - Расширенные настройки безопасности — защита от рейдов, верификация новых участников
2. Возможность продвижения сервера
    - Участие в "Путешествии по серверам" — ваш сервер могут найти новые участники через рекомендации Discord
    - Аналитика вовлечённости — статистика по активности, количеству сообщений, голосовым каналам
3. Улучшенный функционал для участников
    - Специальные роли и бонусы — система наград за активность
    - Кастомные эмодзи и стикеры — доступны для всех участников
    - Звуковая панель — добавление собственных звуков для сервера
4. Поддержка Discord
    - Приоритет в обновлениях — ранний доступ к новым функциям
    - Официальные рекомендации — шанс попасть в топ сообществ вашей тематики

## Как создать сообщество

1. Откройте «Настройки сервера» (правая кнопка мыши по названию сервера).
2. Перейдите во вкладку «Сообщество» → нажмите «Включить».
3. Подтвердите требования:
4. Сервер должен иметь:
    - Правила (канал #правила или #информация).
    - Систему модерации (роли для админов/модераторов).
    - Чёткую структуру каналов (текстовые/голосовые).
