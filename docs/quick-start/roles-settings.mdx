# Настройка ролей и прав

## Для чего нужна настройка ролей

Роли выполняют несколько ключевых функций:

1. Управление доступом:
   - Разграничение прав участников
   - Ограничение доступа к определенным каналам
2. Организация участников:
   - Визуальное выделение групп пользователей (цветные ники, отдельное место в списке участников)
   - Быстрая идентификация команд (модераторы, разработчики, ивент-менеджеры и т. д.)
3. Автоматизация и управление ботом:
   - Некоторые боты используют роли для выдачи прав (например, музыкальные боты требуют роль DJ для управления треками)
   - Автоматическая выдача ролей при выполнении условий (например, после верификации или достижения определенного уровня активности)
4. Геймификация и мотивация:
   - Выдача ролей за активность, участие в ивентах или поддержку сервера (например, «Активный участник», «Легенда сервера»)
   - Роли как награда (например, за победу в турнире или конкурсе)

## Настроить роли

1. Перейдите в «Настройки сервера» (клик правой кнопкой по названию сервера и выберите пункт «Настройки сервера»)
2. В этой вкладке можно изменить название и иконку, добавить описание сервера, настроить уровень модерирования текстового чата
3. Создайте роли (например, "Админ", "Модератор", "Участник")
