# Приглашение участников

Приглашения в Discord нужны для добавления пользователей на сервер, контроля доступа (срок действия, количество использований), защиты от рейдов и ботов, а также для отслеживания, кто и как пригласил новых участников.

## Как создать приглашение

1. В левом меню выберите канал (например, #чат)

2. Нажмите на «Пригласить людей» (или создайте ссылку вручную через «Настройки сервера» → «Приглашения»)

3. Выберите срок действия и ограничения (например, «макс. 100 использований»)

4. Скопируйте ссылку и поделитесь в соцсетях, чатах или с друзьями

![An image from the static](/img/invites.jpg)

:::warning

Если сервер публичный, лучше включить верификацию (например, CAPTCHA), чтобы избежать ботов.

:::
