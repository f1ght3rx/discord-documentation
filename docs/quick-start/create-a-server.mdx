# Создание сервера

## Что такое Discord-сервер?

Discord-сервер – это виртуальное сообщество внутри платформы Discord, где люди могут общаться через текстовые и
голосовые каналы, обмениваться файлами, настраивать роли и автоматизировать процессы с помощью ботов.

__Основные элементы сервера:__

+ Текстовые каналы – для диалога (общие чаты, FAQ, объявления)

+ Голосовые каналы – для живого общения, игр, совещаний

+ Роли и права – гибкая система доступа (админы, модераторы, участники)

+ Боты – автоматизация модерации, музыки, игр и уведомлений

+ Категории – группировка каналов по темам (например, "Игры", "Обучение")

Discord-сервер используется для: совместных игровых сессий, совместного просмотра
видео-контента с синхронизацией через ботов, обсуждения рабочих проектов и удалённого
сотрудничества, проведения онлайн-лекций и образовательных курсов, организации подкастов и голосовых
конференций, совместной работы над творческими проектами.

## Создать Discord-сервер

![An image from the static](/img/add_serv.jpg)

1. Откройте Discord (в браузере, приложении на ПК или мобильном устройстве)
2. В левом боковом меню нажмите на «+» (иконка «Добавить сервер»)
3. Выберите «Создать свой сервер» (или используйте шаблон, если нужно)
4. Введите название сервера и выберите регион (если это важно для голосовых каналов)
5. Нажмите «Создать»

![An image from the static](/img/choose_the_theme.jpg)

Выберите шаблон (например, "Гейминг", "Сообщество").

:::tip

Больше о шаблоне "Сообщество" рассказано в секции [Создание сообщества](../serv-setup/create-a-community.mdx).

:::
