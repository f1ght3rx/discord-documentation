# Создание сервера

## Что такое Discord-сервер?

Discord-сервер представляет собой виртуальное сообщество на платформе Discord, предназначенное для общения через текстовые и голосовые каналы,
обмена файлами, настройки ролей и автоматизации процессов с помощью ботов.

Основные элементы сервера:
    - Текстовые каналы – для диалога (общие чаты, FAQ, объявления)
    - Голосовые каналы – для живого общения, игр, совещаний
    - Роли и права – гибкая система доступа (админы, модераторы, участники)
    - Боты – автоматизация модерации, музыки, игр и уведомлений
    - Категории – группировка каналов по темам (например "Игры", "Обучение")

Discord-серверы активно используются для совместных игровых сессий, коллективного просмотра видео с синхронизацией через ботов,
обсуждения рабочих проектов, удалённого сотрудничества, проведения онлайн-лекций,
образовательных курсов, записи подкастов и организации голосовых конференций, а также для совместной работы над творческими проектами.

## Создать Discord-сервер

![An image from the static](/img/add_serv.jpg)

1. Открыть приложение Discord (в браузере, на ПК или мобильном устройстве)
2. В левом боковом меню нажать на кнопку «+» («Добавить сервер»)
3. Выбрать вариант «Создать свой сервер» или воспользоваться готовым шаблоном
4. Ввести название сервера и, при необходимости, указать регион для голосовых каналов
5. Нажать кнопку «Создать»
