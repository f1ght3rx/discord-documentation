Создание каналов

Создание каналов в Discord — это ключевой способ организовать общение и управлять контентом на сервере. Каналы помогают разделять темы, контролировать доступ и делать сервер удобным для участников.

Как создать канал​

Выберите тип канала(текстовый или голосовой) и рядом с категорией нажмите "+"
